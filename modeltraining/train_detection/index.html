    Object detection model training¶

    YOLOv5¶

    YOLOv5 is probably the easiest to use computer vision framework, including models for object detection, image classification and instance segmentation. You can find a summary of the YOLOv5 detection model architecture here. It is highly recommended to read the tips for best training results if this is your first time training a deep learning model.

    You can use the provided Google Colab notebook for custom training of a:

    • YOLOv5 object detection model

      Includes conversion to .blob format for on-device inference on the Luxonis OAK.

    Check the introduction and features overview, if this is your first time using a Colab notebook.

    After opening the notebook, log in to your Google account and under the File menu choose Save a copy in Drive. Now you can change the code and adapt the training procedure to your use case if desired. Before connecting to a Google Colab cloud instance, make sure to check that GPU is selected as Hardware accelerator under Runtime → Change runtime type.
